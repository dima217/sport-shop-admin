# Sport Admin - Админ-панель для управления складом и заказами

Веб-приложение для кладовщиков и администраторов интернет-магазина спортивных товаров.

## Технологический стек

- **React 19** с TypeScript
- **Vite** - сборщик
- **React Router** - роутинг
- **Zustand** - управление состоянием
- **Material-UI** - UI компоненты
- **Axios** - HTTP клиент
- **React Hook Form** - работа с формами
- **Recharts** - графики и диаграммы
- **date-fns** - работа с датами
- **notistack** - уведомления

## Установка и запуск

### Требования

- Node.js 18+
- Yarn или npm

### Установка зависимостей

```bash
yarn install
# или
npm install
```

### Настройка переменных окружения

Создайте файл `.env` на основе `.env.example`:

```bash
VITE_API_URL=http://84.201.188.209:3000
```

### Запуск в режиме разработки

```bash
yarn dev
# или
npm run dev
```

Приложение будет доступно по адресу `http://localhost:5173`

### Сборка для продакшена

```bash
yarn build
# или
npm run build
```

Собранные файлы будут в папке `dist`

## Функциональность

### Аутентификация
- Вход в систему администратора/кладовщика
- Защита роутов
- Автоматический logout при истечении токена

### Управление категориями
- Просмотр списка категорий
- Создание категории
- Редактирование категории
- Удаление категории

### Управление товарами
- Просмотр списка товаров с фильтрацией и поиском
- Создание товара
- Редактирование товара
- Удаление товара
- Установка скидки на товар
- Удаление скидки

### Управление заказами
- Просмотр списка заказов с фильтрацией по статусу
- Детальная информация о заказе
- Изменение статуса заказа

### Статистика
- Dashboard с общей статистикой
- Графики выручки
- Распределение заказов по статусам
- Статистика по товарам (топ продаж, низкие остатки)

## Структура проекта

```
src/
  ├── components/        # Переиспользуемые компоненты
  │   └── Layout/        # Layout компоненты (Sidebar, Header)
  ├── pages/            # Страницы приложения
  │   ├── Categories/   # Страницы категорий
  │   ├── Products/     # Страницы товаров
  │   └── Orders/       # Страницы заказов
  ├── services/         # API сервисы
  ├── store/            # Zustand stores
  ├── types/            # TypeScript типы
  └── App.tsx           # Главный компонент
```
